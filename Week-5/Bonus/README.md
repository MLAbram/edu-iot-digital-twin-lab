## How to Run & Validate:
**Install Streamlit:**
```bash
pip install streamlit
```

**Launch the App:**
```bash
streamlit run Bonus/dashboard_v2.py
```

**The Test:** Keep the dashboard open on one side of your screen and Wokwi on the other. Move the slider, wait for the Bridge to save it, then hit Refresh (or wait for the auto-refresh) in the browser.
